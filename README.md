# Hotel Booking Analysis Dashboard

## Project Overview

This project analyzes hotel booking data to identify booking trends,
cancellation patterns, customer behavior, and hotel performance. The
project demonstrates a complete data analytics workflow using Python,
SQL, and Power BI.

## Objectives

-   Clean and prepare raw hotel booking data.
-   Perform exploratory data analysis (EDA).
-   Answer business questions using SQL.
-   Build interactive Power BI dashboards.
-   Generate business insights to support decision-making.

## SQL Business Questions

This project answers 21 business questions related to hotel bookings,
customer behavior, cancellations, room allocation, waiting lists, market
segments, customer types, deposit types, ADR, and special requests.

## Key Insights

-   City Hotel received more bookings than Resort Hotel.
-   City Hotel recorded a higher cancellation rate.
-   Online TA generated the highest number of bookings.
-   No Deposit was the most commonly used deposit type.
-   Transient customers represented the largest customer group.
-   City Hotel had significantly more waiting-list bookings than Resort
    Hotel.
